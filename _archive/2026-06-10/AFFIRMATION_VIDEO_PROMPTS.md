# HUDSONSEED — AFFIRMATION VIDEO PROMPT PACK (Veo / SuperGrok)
## RULE: AI-GENERATED / ILLUSTRATIVE CHILDREN ONLY. Never real students. Never implied to be real HudsonSeed kids. Framed as brand animation.

These five 8-second clips are "The HudsonSeed Close" — the affirmation ritual that ends every class.
Generate all five in ONE session so the style stays consistent. Keep every STYLE BLOCK identical; only change the AFFIRMATION line.

====================================================
LOCKED STYLE BLOCK  (paste into EVERY prompt, unchanged)
====================================================
Style: warm, soft, wholesome 3D-animated / Pixar-like illustration. NOT photorealistic — clearly stylized brand animation. Gentle rounded character design. Diverse group of happy young animated children (ages ~6-9), seated calmly. Soft natural window light. Color palette: warm cream walls (#FDF8F0), teal accents (#4A9EA6), soft gold. Calm, safe, joyful mood. Smooth gentle camera, minimal movement. Cozy bright classroom feel. Soft focus background. No text unless specified. No logos. Family-friendly, innocent, reassuring tone.
